Introdução ao HTML

HTML é a sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto). Não é uma linguagem de programação, mas sim uma linguagem de marcação que define a estrutura do seu conteúdo.

O HTML consiste em uma série de elementos que você usa para "embrulhar" ou "marcar" diferentes partes do conteúdo para que ele apareça e aja de uma certa maneira. As tags podem transformar um texto em um link, colocar palavras em itálico, e assim por diante.

Editores de Código

Para escrever código HTML, você precisa de um editor de código. Embora você possa usar um editor de texto simples, é altamente recomendável usar um editor dedicado, que oferece recursos como destaque de sintaxe e autocompletar.

Algumas opções populares e gratuitas são:

  • Visual Studio Code (VS Code): O mais popular atualmente, com milhares de extensões.
  • Sublime Text: Leve, rápido e poderoso.
  • Atom: Criado pelo GitHub, altamente customizável.

HTML Básico

Todo documento HTML precisa ter uma estrutura mínima para que o navegador o entenda corretamente. Essa estrutura é o ponto de partida para qualquer página da web.

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <h1>Meu Primeiro Título</h1>
    <p>Meu primeiro parágrafo.</p>
</body>
</html>

Elementos HTML

Um elemento HTML é geralmente composto por uma tag de abertura, o conteúdo, e uma tag de fechamento. Existem também elementos vazios (sem tag de fechamento), como o <img>.

<p>Este é o conteúdo do elemento.</p>

Atributos HTML

Atributos fornecem informações adicionais sobre um elemento e são sempre especificados na tag de abertura. Eles geralmente vêm em pares de nome/valor como: nome="valor".

<a href="https://www.google.com">Link para o Google</a>
<img src="logo.png" alt="Logotipo do site">

Títulos (Headings)

HTML oferece seis níveis de títulos, de <h1> a <h6>. O <h1> é o mais importante e deve ser usado para o título principal da página, enquanto os outros são subtítulos.

<h1>Título Principal</h1>
<h2>Subtítulo de Nível 2</h2>

Parágrafos

A tag <p> é usada para definir um parágrafo de texto. Navegadores automaticamente adicionam um espaço antes e depois de cada parágrafo.

<p>Este é o primeiro parágrafo.</p>
<p>Este é o segundo parágrafo.</p>

Estilos (Inline CSS)

O atributo style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho, etc. Esta forma (inline) deve ser usada com moderação, preferindo-se o uso de arquivos CSS externos.

<p style="color:red; font-size:18px;">Este parágrafo é vermelho.</p>

Formatação de Texto

O HTML possui vários elementos para formatar texto com um significado especial:

  • <b> - Negrito (sem importância semântica)
  • <strong> - Importante (negrito com importância semântica)
  • <i> - Itálico (sem importância semântica)
  • <em> - Ênfase (itálico com importância semântica)
  • <mark> - Texto marcado
  • <small> - Texto pequeno

Links

A tag <a> define um hyperlink. O atributo mais importante é o href, que indica o destino do link. Por padrão, links aparecem sublinhados e em azul.

<a href="https://easycode.com.br">Visite o Easy Code</a>

Imagens

A tag <img> é usada para incorporar uma imagem em uma página. É uma tag vazia e requer dois atributos: src (o caminho para a imagem) e alt (um texto alternativo para a imagem).

<img src="foto.jpg" alt="Foto de um programador" width="500" height="333">

Tabelas

Tabelas são usadas para exibir dados em formato de linhas e colunas. A estrutura básica envolve as tags <table>, <tr> (linha), <th> (cabeçalho) e <td> (célula de dado).

<table>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>João</td>
<td>25</td>
</tr>
</table>

Listas

HTML suporta listas não ordenadas (<ul>, com marcadores) e listas ordenadas (<ol>, com números). Cada item da lista é definido pela tag <li>.

<h2>Lista Não Ordenada</h2>
<ul>
<li>Café</li>
<li>Leite</li>
</ul>

<h2>Lista Ordenada</h2>
<ol>
<li>Primeiro passo</li>
<li>Segundo passo</li>
</ol>

Classes e ID

Os atributos class e id são usados para apontar para elementos específicos no CSS e JavaScript.

  • O atributo class pode ser usado em múltiplos elementos para aplicar o mesmo estilo.
  • O atributo id deve ser único na página e é usado para identificar um único elemento.
<!-- Usando uma classe -->
<p class="texto-destaque">Este parágrafo tem destaque.</p>

<!-- Usando um ID -->
<h1 id="titulo-principal">Título Único</h1>

Layout Semântico

HTML5 introduziu novos elementos semânticos que descrevem claramente suas partes para o navegador e para os desenvolvedores, melhorando a acessibilidade e o SEO.

  • <header> - Define um cabeçalho para um documento ou seção.
  • <nav> - Define um conjunto de links de navegação.
  • <section> - Define uma seção em um documento.
  • <article> - Define conteúdo independente e autocontido.
  • <aside> - Define conteúdo à parte do conteúdo principal (como uma barra lateral).
  • <footer> - Define um rodapé para um documento ou seção.

Introdução ao CSS

CSS (Cascading Style Sheets) é a linguagem que usamos para estilizar e dar vida ao nosso HTML. Ele controla cores, fontes, espaçamentos, layouts e muito mais, separando a apresentação da estrutura do conteúdo.

Sintaxe CSS

Uma regra CSS é composta por um seletor e um bloco de declaração. O seletor aponta para o elemento HTML que você quer estilizar. O bloco contém uma ou mais declarações (propriedade e valor).

h1 {
color: blue;
font-size: 24px;
}

Seletores CSS

Seletores definem a quais elementos nossos estilos serão aplicados. Os básicos são: seletor de elemento (p), de classe (.minha-classe) e de ID (#meu-id).

p { font-family: sans-serif; }
.alerta { color: red; }
#logo { width: 150px; }

Como Usar CSS

A forma mais recomendada é usar um arquivo CSS externo, linkado no <head> do HTML: <link rel="stylesheet" href="style.css">.

Cores

Cores em CSS podem ser definidas por nomes (red), códigos HEX (#FF0000), RGB (rgb(255,0,0)) ou RGBA (para transparência).

p {
color: #333333; /* Cor do texto */
background-color: lightblue; /* Cor de fundo */
}

Backgrounds

Além da cor, você pode definir uma imagem de fundo e controlar sua repetição, posição e se ela rola com a página.

body {
background-image: url("fundo.jpg");
background-repeat: no-repeat;
background-position: center;
background-size: cover;
}

Bordas

A propriedade border permite especificar a largura, o estilo (solid, dotted, etc.) e a cor da borda de um elemento.

div {
border: 1px solid #ccc;
}

Margens

A propriedade margin define o espaçamento externo de um elemento, criando espaço ao redor dele.

.caixa {
margin: 20px; /* 20px em todos os lados */
}

Padding

A propriedade padding define o espaçamento interno de um elemento, entre a borda e o conteúdo.

.caixa {
padding: 20px; /* 20px em todos os lados */
}

CSS Box Model

O Box Model (Modelo de Caixa) é um dos conceitos mais fundamentais do CSS. Ele dita que cada elemento HTML na sua página é uma caixa retangular. Essa caixa é composta por quatro camadas:

  • Conteúdo (Content): A área onde seu texto e imagens aparecem.
  • Preenchimento (Padding): Uma área transparente que envolve o conteúdo, criando espaço interno.
  • Borda (Border): A linha que fica ao redor do padding e do conteúdo.
  • Margem (Margin): Uma área transparente que fica do lado de fora da borda, criando espaço entre esta e outras caixas.

Pense nele como uma moldura de quadro: o conteúdo é a foto, o padding é o paspatur (a cartolina entre a foto e a moldura), a borda é a moldura em si, e a margem é o espaço entre este quadro e outros na parede.

.exemplo-box-model {
width: 300px;
padding: 20px;
border: 5px solid #007BFF;
margin: 30px;
background-color: #1a1a2e;
}
/* A largura total na tela será:
300px (width) + 40px (padding) + 10px (border) = 350px
A margem não entra no cálculo do tamanho do elemento, apenas o afasta dos outros. */

Dica: Para facilitar os cálculos, muitos desenvolvedores usam box-sizing: border-box;. Com isso, o padding e a borda são incluídos na largura (width) e altura (height) totais do elemento, em vez de serem somados a elas.

Texto

CSS oferece um controle refinado sobre a aparência do texto. As propriedades mais comuns são:

  • text-align: Alinha o texto horizontalmente (left, right, center, justify).
  • text-decoration: Adiciona ou remove decorações como sublinhado (underline) ou linha através (line-through). Use none para remover o sublinhado de links.
  • text-transform: Transforma o texto para maiúsculas (uppercase), minúsculas (lowercase) ou apenas a primeira letra de cada palavra em maiúscula (capitalize).
  • line-height: Define a altura da linha, controlando o espaçamento vertical entre as linhas de um texto.
  • letter-spacing: Controla o espaço entre os caracteres.
.paragrafo-estilizado {
text-align: center;
text-transform: uppercase;
line-height: 1.8;
letter-spacing: 1px;
color: #007BFF;
}

Fontes

Com CSS, você controla a família da fonte (font-family), o peso (font-weight), o tamanho (font-size) e o estilo (font-style).

  • font-family: Especifica a fonte para um elemento. É uma boa prática fornecer uma lista de fontes "fallback" (alternativas), terminando com uma família genérica (como sans-serif).
  • font-size: Define o tamanho do texto.
  • font-weight: Define o "peso" ou a "grossura" da fonte (ex: normal, bold, ou valores numéricos como 400, 700).
  • font-style: Define o estilo da fonte, mais comumente usado para italic (itálico).
.texto-com-fonte {
font-family: 'Roboto', Arial, sans-serif;
font-size: 18px;
font-weight: bold; /* ou 700 */
font-style: italic;
}

Dica: Use o Google Fonts para importar centenas de fontes gratuitas e de alta qualidade para o seu site com uma única linha de código no seu HTML ou CSS.

Links

Você pode estilizar os diferentes estados de um link usando pseudo-classes. Isso permite que a aparência de um link mude quando o usuário interage com ele.

  • a:link: O estado normal de um link que ainda não foi visitado.
  • a:visited: Um link que o usuário já visitou.
  • a:hover: Quando o usuário passa o cursor do mouse sobre o link.
  • a:active: O momento exato em que o link está sendo clicado.
a {
text-decoration: none; /* Remove o sublinhado padrão */
color: #007BFF;
transition: color 0.2s; /* Adiciona uma transição suave na cor */
}

a:visited {
color: #8a2be2; /* Roxo para links já visitados */
}

a:hover {
color: #00c3ff; /* Um azul mais claro ao passar o mouse */
text-decoration: underline; /* Adiciona sublinhado no hover */
}

a:active {
color: #ff4500; /* Laranja enquanto está sendo clicado */
}

Listas

A propriedade list-style-type permite alterar o marcador de uma lista. Você pode usar círculos, quadrados, números romanos, ou até mesmo remover completamente os marcadores.

/* Para listas não ordenadas */
ul.quadrados {
list-style-type: square;
}

/* Para listas ordenadas */
ol.romanos {
list-style-type: upper-roman;
}

/* Para remover os marcadores */
ul.sem-marcador {
list-style-type: none;
padding-left: 0; /* Remove o padding padrão para alinhar à esquerda */
}

Tabelas

Estilizar tabelas geralmente envolve adicionar bordas, padding para as células e usar border-collapse para um visual mais limpo.

table.customizada {
width: 100%; /* Faz a tabela ocupar toda a largura disponível */
border-collapse: collapse; /* Une as bordas das células */
}

table.customizada th, table.customizada td {
border: 1px solid #2a2a4a; /* Borda para cada célula */
padding: 12px; /* Espaçamento interno nas células */
text-align: left; /* Alinha o texto à esquerda */
}

table.customizada th {
background-color: #1a1a2e; /* Cor de fundo para o cabeçalho */
color: #ffffff;
}

/* Efeito de "zebra" para melhor legibilidade */
table.customizada tr:nth-child(even) {
background-color: #161625;
}

Display

A propriedade display é uma das mais importantes do CSS para controle de layout. Ela define como um elemento se comporta e interage com os outros.

  • display: block;: O elemento ocupa toda a largura disponível, com uma quebra de linha antes e depois. Exemplos: <div>, <p>, <h1>.
  • display: inline;: O elemento ocupa apenas o espaço necessário para seu conteúdo e não força uma quebra de linha. Não é possível definir width ou height. Exemplos: <span>, <a>, <strong>.
  • display: inline-block;: Um híbrido. O elemento flui com o texto como um elemento inline, mas aceita width, height, margin e padding como um elemento block.
  • display: flex;: Ativa o layout Flexbox, um modelo de layout unidimensional poderoso para alinhar e distribuir espaço entre itens em um contêiner.
  • display: none;: O elemento é completamente removido da página, como se não existisse no HTML.
            
                .container-flex {
                display: flex;
                justify-content: space-around; /* Distribui os itens com espaço ao redor */
                background-color: #1a1a2e;
                padding: 10px;
                }
                .item {
                width: 100px;
                height: 50px;
                background-color: #007BFF;
                }

Position

A propriedade position especifica o método de posicionamento usado para um elemento. Ela é usada em conjunto com as propriedades top, right, bottom e left.

  • static: O valor padrão. O elemento segue o fluxo normal da página.
  • relative: O elemento é posicionado em relação à sua posição normal. Você pode movê-lo com top/left, mas o espaço original que ele ocupava é preservado.
  • absolute: O elemento é posicionado em relação ao ancestral posicionado mais próximo (ou ao corpo da página). Ele é removido do fluxo normal do documento.
  • fixed: O elemento é posicionado em relação à viewport (a janela do navegador). Ele não se move, mesmo quando a página é rolada. Ótimo para headers fixos ou botões de "voltar ao topo".
.container-relativo {
                position: relative;
                width: 300px;
                height: 200px;
                border: 2px solid #ccc;
                }

                .box-absoluta {
                position: absolute;
                bottom: 10px;
                right: 10px;
                width: 80px;
                height: 40px;
                background-color: #ff4500;
                }
/* A .box-absoluta ficará a 10px da parte inferior e 10px da direita
DENTRO do seu .container-relativo */

Z-index

A propriedade z-index especifica a ordem de empilhamento (profundidade) de um elemento. Ela só funciona em elementos que têm uma propriedade position diferente de static.

Um elemento com um valor de z-index maior ficará na frente de um elemento com um valor menor.


            .caixa-vermelha {
            position: absolute;
            top: 20px;
            left: 20px;
            width: 150px;
            height: 150px;
            background-color: rgba(255, 0, 0, 0.7);
            z-index: 1; /* Fica atrás */
            }

            .caixa-azul {
            position: absolute;
            top: 60px;
            left: 60px;
            width: 150px;
            height: 150px;
            background-color: rgba(0, 123, 255, 0.7);
            z-index: 2; /* Fica na frente */
            }

Overflow

A propriedade overflow controla o que acontece quando o conteúdo de um elemento é maior do que a sua caixa.

  • visible (padrão): O conteúdo "vaza" para fora da caixa.
  • hidden: O conteúdo que vaza é cortado e fica invisível.
  • scroll: O conteúdo é cortado, mas barras de rolagem são adicionadas para que o usuário possa ver o resto.
  • auto: Adiciona barras de rolagem apenas quando necessário (se o conteúdo vazar).

            .caixa-com-overflow {
            width: 250px;
            height: 100px;
            border: 1px solid #ccc;
            padding: 10px;
            overflow: auto; /* Melhor opção na maioria dos casos */
            }

Unidades

CSS usa várias unidades para expressar tamanhos. Elas se dividem em duas categorias principais: absolutas e relativas.

  • Absolutas: São fixas e não mudam de tamanho. A mais comum é o px (pixel).
  • Relativas: São flexíveis e seu tamanho depende de outro valor. São essenciais para criar layouts responsivos.
    • %: Relativo ao elemento pai.
    • em: Relativo ao tamanho da fonte do elemento atual.
    • rem: (root em) Relativo ao tamanho da fonte do elemento raiz (<html>). É a mais recomendada para fontes.
    • vw / vh: 1% da largura (viewport width) ou altura (viewport height) da tela.
            
            html {
            font-size: 16px; /* Base para o rem */
            }

            h1 {
            font-size: 2rem; /* 2 * 16px = 32px */
            }

            p {
            font-size: 1rem; /* 1 * 16px = 16px */
            }

            .container-responsivo {
            width: 80vw; /* Ocupa 80% da largura da tela */
            margin: 0 auto; /* Centraliza */
            }

Especificidade

A especificidade é o conjunto de regras que os navegadores usam para decidir qual estilo CSS aplicar quando múltiplas regras apontam para o mesmo elemento e entram em conflito. A regra com maior especificidade vence.

A hierarquia, da mais forte para a mais fraca, é a seguinte:

  1. Estilos Inline: (ex: <p style="color: red;">) - Mais forte de todas.
  2. ID: (ex: #meu-id)
  3. Classes, pseudo-classes, seletores de atributo: (ex: .minha-classe, :hover, [type="submit"])
  4. Seletores de elemento e pseudo-elementos: (ex: p, ::before)
<!-- HTML -->
<p id="paragrafo-unico" class="texto">Qual cor eu terei?</p>

<!-- CSS -->
p { color: blue; }         /* Especificidade: 1 */
.texto { color: green; }   /* Especificidade: 10 */
#paragrafo-unico { color: red; } /* Especificidade: 100 */

/* O parágrafo será VERMELHO, pois o seletor de ID é o mais específico. */
